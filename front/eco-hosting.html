Stack técnico

Tecnología que trabaja menos, rinde más

Cada componente del stack fue elegido por ser el mejor en su función y el más eficiente energéticamente.

Servidor web

LiteSpeed Web Server

El servidor web más rápido para PHP y WordPress disponible en hosting compartido. Maneja carga concurrente con un tercio de los recursos de Apache, tiene su propio sistema de caché integrado y es compatible 100% con configuraciones .htaccess.

9× más rápido que Apache HTTP/3 + QUIC LSCache nativo para WP
Caché objetos

Redis Object Cache

WordPress hace cientos de consultas a la base de datos en cada carga de página. Redis mantiene los resultados en memoria RAM, eliminando hasta el 90% de esas consultas. El resultado: respuestas de base de datos en microsegundos, no milisegundos.

Hasta 90% menos DB queries Respuesta en µs Persistencia en RAM
CDN global

Cloudflare CDN

Los assets estáticos (imágenes, CSS, JS) se sirven desde el nodo de Cloudflare más cercano al visitante. Latencia mínima independientemente de su ubicación. Con protección DDoS Layer 7 y WAF incluidos, sin configuración adicional.

+300 nodos globales WAF + DDoS L7 SSL gratis

MariaDB + MySQL optimizado

Instancias de MariaDB con configuración específica para WordPress: buffer pools ajustados, query cache activo, y control de tablas wp_options para evitar el autoload inflado, que es uno de los mayores cuellos de botella en WordPress de alto tráfico.

Preguntas frecuentes

Lo que nos preguntan sobre hosting

Respuestas directas sobre planes, migración y rendimiento.

¿Qué es el ECO Hosting de Replanta?

Un servicio de alojamiento web de alto rendimiento con energía 100% renovable certificada. Usa LiteSpeed como servidor web, Redis como caché de objetos y Cloudflare CDN para la entrega de contenido desde el nodo más cercano al visitante. Combina máximo rendimiento con mínima huella ambiental — sin compromisos entre uno y otro.

¿El hosting de Replanta funciona bien para WordPress?

Está optimizado específicamente para WordPress y WooCommerce. LiteSpeed incluye un plugin de caché propio extremadamente eficiente (LSCache), Redis gestiona la caché de objetos de WordPress y Cloudflare CDN entrega los assets desde el nodo más cercano. Muchos clientes consiguen TTFB inferior a 150ms y PageSpeed por encima de 95 en móvil y escritorio.

¿La migración está incluida?

Sí. Realizamos la migración completa — archivos, base de datos, configuración de DNS — sin coste adicional y sin tiempo de inactividad. El proceso habitual dura entre 2 y 4 horas. Solo necesitas darnos acceso a tu hosting actual (cPanel, SFTP o similar) y nosotros nos encargamos del resto.

¿Puedo alojar varios WordPress en un mismo plan?

El Plan Sauce soporta 1 sitio e incluye 5 cuentas de email. Los planes Roble y Cedro admiten sitios ilimitados. El Plan Cedro es ideal para agencias o carteras de proyectos — y podemos configurar paneles de cliente individuales. Todos los planes incluyen staging.

¿Dónde están los servidores?

Contamos con dos centros de datos: Londres (Europa) para clientes de la UE, y Florida (EE.UU.) para clientes de Latinoamérica. Cloudflare CDN distribuye el contenido desde más de 300 nodos globales para minimizar la latencia en cualquier región. Energía renovable certificada en ambos.

¿Qué incluyen los backups?

Backups diarios automáticos gestionados por Backuply, almacenados en ubicación externa (no en el mismo servidor). Los backups incluyen archivos y base de datos completa. Puedes solicitar una restauración a cualquier punto disponible abriendo un ticket de soporte.

¿El hosting incluye certificado SSL?

Sí. Let's Encrypt SSL automático en todos los planes, con renovación automática sin coste. También habilitamos el forzado de HTTPS y los encabezados de seguridad recomendados (HSTS, X-Frame-Options, X-Content-Type-Options) en todos los sitios desde el primer día.

¿Qué pasa si mi web crece en tráfico?

Puedes escalar de plan sin perder configuración ni datos. El cambio de plan se realiza en pocas horas, sin tiempo de inactividad. Para proyectos con picos de tráfico imprevisibles o con requisitos de alta disponibilidad, también ofrecemos infraestructura VPS gestionada — consúltanos.

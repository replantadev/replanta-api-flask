# Replanta Care

Plugin de mantenimiento WordPress automatico para clientes de Replanta. Gestiona actualizaciones, backups, seguridad y reportes de forma invisible para el usuario final.

**Version:** 1.14.3 | **Requiere:** WordPress 6.0+, PHP 7.4+ | **Licencia:** Privada (license key requerido)

---

## Que hace Replanta Care

Replanta Care instala un sistema de mantenimiento automatico en el sitio WordPress del cliente. Todas las tareas se ejecutan en segundo plano via Action Scheduler sin afectar al rendimiento del sitio.

| Area | Que automatiza |
|---|---|
| Actualizaciones | WordPress core, plugins y temas con backup previo y rollback |
| Seguridad | Escaneo de vulnerabilidades, cambios de archivos, logins fallidos |
| Rendimiento | Core Web Vitals, WPO, cache |
| Contenido | Medios huerfanos, errores 404, SEO basico |
| Reportes | Informe semanal/mensual al cliente |
| Integraciones | Cloudflare, Backblaze B2, Replanta Hub |

## Planes

| Plan | Para quien | Tareas incluidas |
|---|---|---|
| **Semilla** | Sites sencillos | Actualizaciones + health check + reporte mensual |
| **Raiz** | Blogs y corporativos | Semilla + seguridad + WPO + 404 |
| **Ecosistema** | Tiendas y proyectos criticos | Raiz + CWV + anomalias + staging + backup avanzado |
